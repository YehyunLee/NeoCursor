# SilentCursor

## Problem Statement
Whether you're deep in a coding session or drafting long-form content, constantly typing, clicking, or even speaking aloud can lead to fatigue. Traditional speech-to-text tools still require audible speech, which can be draining and impractical in shared or quiet environments. Users need a hands-free, low-effort way to interact with their computers without compromising productivity.

## Solution
SilentCursor enables fully hands-free computing by combining eye-tracking mouse control with silent speech input. By leveraging computer vision, visual speech recognition, and large language models, the standalone desktop application lets users navigate interfaces with subtle eye movements and type through silent speech articulation—no physical movement or audible voice required.
